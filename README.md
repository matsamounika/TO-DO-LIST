# Todo List App

## 📌 About Project
This is a simple Todo List web application built using HTML, CSS, and JavaScript.

## 🚀 Features
- Add new tasks
- Delete tasks
- Mark tasks as completed
- Data saved using localStorage

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript

## 🎯 What I learned
- DOM manipulation
- Event handling
- LocalStorage usage
